# O que é e como usar Margin, Padding e Broder   

## Border
- A Border é a linha que fica ao redor do conteúdo e do preenchimento. Ela serve para criar limites visuais.

**Como usar:**
A linha da caixa.

***Exemplo:***
border: 2px solid black;

## Margin
- A Margin é o espaço fora da borda. Ela serve para afastar um elemento de outro. Diferente do padding, a margem é sempre transparente.

**Como usar:**
Espaço entre essa caixa e a caixa vizinha.

***Exemplo:***
margin: 50px;

## Padding
- As propriedades de preenchimento (padding) do CSS são usadas para gerar espaço ao redor do conteúdo de um elemento, dentro de quaisquer bordas definidas.

**Como usar:**
Espaço entre o texto e a linha da caixa.

***Exemplo:***
padding: 20px;
